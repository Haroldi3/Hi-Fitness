# FlexIQ Mobile (Expo)

Use this folder as a drop-in starter.

## Steps
1) Create an Expo app:
   npx create-expo-app flexiq-mobile

2) Copy these into your new Expo app:
   - Replace App.js with this App.js
   - Copy the entire src/ folder

3) Install deps:
   npm i axios @react-native-async-storage/async-storage @react-native-picker/picker
   npm i @react-navigation/native @react-navigation/bottom-tabs
   npx expo install react-native-screens react-native-safe-area-context

4) Set API_BASE in src/api/client.js
